Bind Anvil components to Tableau filters and parameters

In order to keep Anvil components in sync with components on the dashboard (like filters and parameters), use the register_event_handler method of a Dashboard object to set the properties of an Anvil component. In one sentence, we are saying: “Whenever the filter changes on the Tableau dashboard, go and upate this Anvil component”.

For this how-to we’ll assume there is a single Anvil form that contains some labels (prepended with lbl_) and a single button with a function named btn_change_param_click bound to its click event. There is also a text box named text_box_1 that will allow a user to input a new value for the parameter.

 1 # Form code for a Form named "BindComponents"
 2 
 3 from ._anvil_designer import BindComponentsTemplate
 4 from anvil import tableau
 5 from tableau_extension.api import get_dashboard
 6 
 7 class BindComponents(BindComponentsTemplate):
 8   def __init__(self, **properties):
 9     self.init_components(**properties)
10     self.dashboard = get_dashboard()
11 
12     # Register event handlers
13     self.dashboard.register_event_handler('filter_changed', self.filter_change)
14     self.dashboard.register_event_handler('parameter_changed', self.param_change)
15 
16     # "myparam" is the name of the parameter
17     self.param = self.dashboard.get_parameter('myparam')
18 
19     # Get the first filter on the dashboard
20     self.filter = self.dashboard.filters[0]
21 
22   def filter_change(self, event):
23     """ This function runs whenever a Tableau filter changes. """
24     self.lbl_filter_name.text = event.filter.field_name
25     self.lbl_filter_value.text = self.filter.applied_values
26 
27   def param_change(self, event):
28     """ This function runs whenever a Tableau parameter changes. """
29     self.lbl_param_name.text = event.parameter.name
30     self.lbl_param_value.text = self.param.value
31 
32   def btn_change_param_click(self, **event_args):
33     """ This function is bound to the click event of the 'Apply' button and
34     updates a parameter in Tableau from Anvil. """
35     if self.text_box_1.text:
36       self.param.value = self.text_box_1.text

The filter_changed and parameter_changed events are registered to methods that simply update text labels in Anvil. i.e.

self.lbl_filter_name.text = event.filter.field_name

To update the Tableau parameter from Anvil, we assign the value property of the parameter in the click event handler (btn_change_param_click) for an Anvil button.

self.param.value = self.text_box_1.text
