Bind a tableau categorical filter to an Anvil component

This guide will show how to build a repeating panel in Anvil that contains a row for each value selected in a categorical filter. It assumes that the Anvil app contains a single repeating panel named repeating_panel_1.

 1 from ._anvil_designer import MainTemplate
 2 from anvil import *
 3 
 4 from tableau_extension import api
 5 
 6 class Main(MainTemplate):
 7   def __init__(self, **properties):
 8     self.init_components(**properties)
 9 
10     self.dashboard = api.get_dashboard()
11     self.dashboard.register_event_handler('filter_changed', self.show_values)
12 
13   def show_values(self, event):
14     """ This method runs whenever a filter is changed. """
15     filter_values = event.filter.applied_values
16 
17     print(f'{event.filter.field_name} filter was changed to {filter_values}.')
18 
19     self.repeating_panel_1.items = filter_values
20     self.refresh_data_bindings()

As always, register an event handler to the dashboard object and set the callback function (show_values).

To retrieve the values of the filter that are currently applied, use applied_values. In the callback function, set the repeating panel’s repeating_panel_1.items to the applied_values of the filter.
